Интерактивное веб-приложение для студентов ОГУ (Оренбургский Государственный Университет), создающее персональную статистику за учебный год в формате "историй", аналогично Spotify Wrapped.

## О проекте

osu-recap визуализирует ваши учебные данные — посещаемость, оценки, время, проведенное в университете, и другие метрики — превращая сухие цифры в увлекательную анимационную историю. Проект создан студентами для студентов, чтобы сделать анализ учебного года интересным и наглядным.

## Безопасность и конфиденциальность

Мы понимаем важность защиты ваших личных данных, особенно логина и пароля от университетского аккаунта.

**Почему нам можно доверять:**

*   **Локальная обработка:** Ваши учетные данные (логин и пароль) используются **исключительно на вашем устройстве**. Приложение работает как клиент, отправляя запросы напрямую к серверам университета.
*   **Отсутствие сторонних баз данных:** Мы не сохраняем, не передаем и не логируем ваши данные на сторонних серверах. Вся статистика генерируется в реальном времени в вашем браузере.
*   **Открытый исходный код:** Вы можете самостоятельно изучить код проекта в этом репозитории и убедиться в отсутствии скрытых механизмов сбора данных.

## Технологический стек

*   **Frontend:** React, TypeScript, Vite
*   **Стилизация:** Tailwind CSS
*   **UI Компоненты:** Radix UI, shadcn/ui
*   **Анимации:** Motion (Framer Motion)
*   **Иконки:** Lucide React

## Запуск проекта

Для запуска приложения вам потребуется установленный [Node.js](https://nodejs.org/).

1.  **Клонируйте репозиторий:**

    ```bash
    git clone https://github.com/osu-platform/osu-recap.git
    cd osu-recap
    ```

2.  **Установите зависимости:**

    ```bash
    npm install
    ```

3.  **Запустите режим разработки:**

    ```bash
    npm run dev
    ```

    Приложение будет доступно по адресу, указанному в терминале (обычно http://localhost:5173).

4.  **Сборка для продакшена:**

    ```bash
    npm run build
    ```
